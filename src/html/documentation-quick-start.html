In this quick start, we will:

  • create a system,
  • run it client side
  • and server side.

Create a system

  • click on the + button on the left toolbar,
  • set StarWars as name,
  • a new system is now created and appears in the Systems list.

Create a schema

  • click on the Schema tab,
  • click on the + button on the left toolbar,
  • set Jedi as name,
  • a new schema is now created and appears in the Schemas list,
  • click on it,
  • update the MSON with this:
{\n
  "_name": "Jedi",\n
  "_inherit": [\n
    "RuntimeComponent"\n
  ],\n
  "firstName": "property",\n
  "lastName": "property",\n
  "father": "link",\n
  "fullName": "method"\n
}
  • save.

note:

Use property for creating a property.
Use link for creating a link between models.
Use collection for creating a collection.
Use method for creating a method.
Use event for creating an event.

Create a model

  • click on the Models tab,
  • click on the + button on the left toolbar,
  • set Jedi as name and select JediS as schema,,
  • a new model is now created and appears in the Models list.

The model is generated with the informations you set in the schema.
You can edit these informations by editing the model.

Create a behavior

  • click on the Behaviors tab,
  • select Jedi on right,
  • click on the + button on the left toolbar,
  • select fullName,
  • a new behavior is now created and appears in the Behaviors list,
  • click on it,
  • update the code with this:
function fullName(param) {\n
  var result = this.firstName() + ' ' + this.lastName();\n
  return result;\n
}
  • save.

Create components

  • click on the Components tab,
  • select Jedi on right,
  • click on the + button on the left toolbar,
  • a new component is now created and appears in the Components list,
  • click on it,
  • update the JSON with this:
{\n
  "_id": "13331191e31d2d0",\n
  "father": {},\n
  "firstName": "Dark",\n
  "lastName": "Vador"\n
}
  • save.
  • click on the + button on the left toolbar,
  • a new component is now created and appears in the Components list,
  • click on it,
  • update the JSON with this:
{\n
  "_id": "166c21e4a01dd30",\n
  "father": "13331191e31d2d0",\n
  "firstName": "Luke",\n
  "lastName": "Skywalker"\n
}
  • save.

Add code on the main method of the system

  • click on the Behaviors tab,
  • click on StarWars on the right list,
  • click on the element who has got 'main' as name,
  • update the code with this:
function main() {\n 
   var luke = null;\n
\n
    // get the component\n
    luke = this.require('166c21e4a01dd30');\n
\n    
// get the name of the father of Luke\n
    console.log(luke.father().fullName());\n
}
  • save.

Test the system

  • click on the run button on the left of toolbar,
  • open the console of the browser and you will see the result.

Run your system client side

  • click on the export button,
  • select HTML,
  • your system will be exported in a HTML file,
  • load the page.

Run your system server side

  • click on the export button,
  • select JS (node),
  • your system will be exported in a JavaScript file,
  • save the file on your server project,
  • in your project, install this package to run your system:
npm install system-runtime --save
  • run your system:
node StarWars.js

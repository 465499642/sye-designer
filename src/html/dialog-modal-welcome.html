System Designer will help you to design your application before coding.
To do so System Designer embraces the System Driven Development methodology, it means that you have to follow these steps to create your application:

Simple steps:

1. create a system,
2. add some code in the main behavior of your system and
3. run your system.

Normal steps:

1. create a system,
2. create a schema,
3. create a model that implements your schema,
4. create behaviors,
5. create components,
6. add some code in the main behavior of your system,
7. run your system,
8. open the console of your browser and
9. use SyrupJS apis to manage your system.
